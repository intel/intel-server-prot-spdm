aardvark.smb_avark

platform:

Linux, Windows

synopsis:

i2c driver wrapper to send and receive traffic to CPLD using aadvark tool

Aardvark is I2C/SPI host adapter, refer https://www.totalphase.com/products/aardvark-i2cspi/

CPLD slave address is 0x70
